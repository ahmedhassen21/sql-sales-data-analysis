# Sales Data Analysis Project (SQL)
SQL | Data Analysis | Portfolio Project

## Project Overview

This project analyzes sales data using SQL to extract valuable business insights.
The analysis includes data exploration, customer behavior analysis, product performance evaluation, and revenue insights.

The goal of this project is to demonstrate SQL skills used by data analysts in real-world business scenarios.

---

## Database Schema

The analysis uses the following tables:

* **fact_sales** → transaction level sales data
* **dim_customers** → customer information
* **dim_products** → product details

---

```
Scripts

├── eda_analysis.sql

```

Each SQL script focuses on a different analytical task.

---

## Key Analysis Areas

### 1. Data Exploration

* Explore database tables and columns
* Understand customer distribution by country and gender
* Analyze product hierarchy

### 2. Sales Performance Analysis

* Monthly sales trends
* Running totals
* Moving average calculations

### 3. Product Performance

* Compare product sales vs average performance
* Identify yearly growth or decline

### 4. Customer Analysis

* Customer segmentation (VIP, Regular, New)
* Customer lifespan and spending behavior

### 5. Revenue Analysis

* Revenue by category
* Revenue by customer
* Revenue distribution by country

### 6. Top / Bottom Performance

* Top selling products
* Worst performing products
* Top customers by revenue
